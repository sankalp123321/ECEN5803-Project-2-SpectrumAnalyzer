Detailed Description

Group one or more functions in an instance of a CallChain, then call them in sequence using CallChain::call(). Used mostly by the interrupt chaining code, but can be used for other purposes.

Deprecated:
Do not use this class. This class is not part of the public API of mbed-os and is being removed in the future.
Note
Synchronization level: Not protected

Example:

#include "mbed.h"
CallChain chain;
void first(void) {
printf("'first' function.\n");
}
void second(void) {
printf("'second' function.\n");
}
class Test {
public:
void f(void) {
printf("A::f (class member).\n");
}
};
int main() {
Test test;
chain.add(second);
chain.add_front(first);
chain.add(&test, &Test::f);
chain.call();
}
