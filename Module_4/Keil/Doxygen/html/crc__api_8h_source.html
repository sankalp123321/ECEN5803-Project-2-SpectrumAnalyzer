211/* Reads the checksum result from the CRC module.
212 *
213 * Reads the final checksum result for the final checksum value. The returned
214 * value is cast as an unsigned 32-bit integer. The actual size of the returned
215 * result depends on the polynomial used to configure the CRC module.
216 *
217 * Additional transformations that are used in the default configuration of the
218 * input polynomial are applied to the result before it is returned from this
219 * function. These transformations include: the final xor being appended to the
220 * calculation, and the result being reflected if required.
221 *
222 * Calling this function multiple times is undefined. The first call to this
223 * function returns the final result of the CRC calculation. The return
224 * value on successive calls is undefined because the contents of the register after
225 * accessing them is platform-specific.
226 *
227 * This function is not thread safe. There is only one instance of the CRC
228 * module active at a time. Calling this function from multiple contexts may
229 * return incorrect data or affect the current state of the module.
230 *
231 * \return The final CRC checksum after the reflections and final calculations
232 * have been applied.
233 */
234uint32_t hal_crc_get_result(void);
